# Usage


### Send Text message
```python
import discum
bot = discum.Client('Token')
bot.sendMessage("383003333751856129","Hello You :)")
```


### Check if token is vaild
```python
import discum
bot = discum.Client('Token')
bot.connectionTest()
```
